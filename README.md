# VitaClínica — Landing Page

Site estático de apresentação da clínica VitaClínica, desenvolvido com HTML e CSS puro.

## 🚀 Como executar

### Opção 1 — Abrir direto no navegador (mais simples)
1. Baixe ou clone este repositório
2. Localize o arquivo `index.html`
3. Dê dois cliques no arquivo — ele abrirá no seu navegador

### Opção 2 — Via Live Server (VS Code)
1. Instale a extensão **Live Server** no VS Code
2. Abra a pasta do projeto no VS Code
3. Clique com o botão direito no `index.html`
4. Selecione **"Open with Live Server"**

### Opção 3 — Via terminal (Python)
```bash
# Clone o repositório
git clone https://github.com/Djams1/TrabalhoPWEB.git
cd TrabalhoPWEB

# Inicie um servidor local
python -m http.server 8080
```
Acesse no navegador: `http://localhost:8080`

## 📋 Requisitos
- Nenhuma dependência ou instalação necessária
- Funciona em qualquer navegador moderno (Chrome, Firefox, Edge, Safari)

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3 (variáveis CSS, Grid, Flexbox)
- Google Fonts (Playfair Display, DM Sans)
